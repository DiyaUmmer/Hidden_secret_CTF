# Hidden Secret - Web Exploitation CTF

A Flask-based cybersecurity Capture The Flag (CTF) platform designed to demonstrate web exploitation concepts, hidden clue discovery, and interactive challenge development.

---

## Features

* Multi-level web exploitation challenges
* Hidden source code clues
* Token-based progression system
* Interactive puzzle-solving
* Cybersecurity themed gameplay
* Flask backend integration
* Responsive frontend design

---

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite

---

## Skills Demonstrated

* Web application development
* Cybersecurity challenge creation
* Flask backend development
* CTF design
* Source code inspection concepts
* Secure coding practices
